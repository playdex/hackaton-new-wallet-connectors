Implemented sandbox with new wallet connectors based on wagmi:
* [Native wagmi](https://wagmi.sh/);
* [Rainbow-kit](https://www.rainbowkit.com/);
* [Family kit](https://family.co/);

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

[API routes](https://nextjs.org/docs/api-routes/introduction) with wallet connectors can be accessed on [wagmi](http://localhost:3000/wagmi) [rainbow](http://localhost:3000/rainbow). These endpoints can be reached from [main page](http://localhost:3000/).
